# Platform Checklist

## YouTube Shorts

- [ ] Video is vertical 9:16.
- [ ] Video length is suitable for Shorts and under 3 minutes.
- [ ] `youtube_title` is present and not misleading.
- [ ] `youtube_description` is present.
- [ ] `youtube_tags` are included.
- [ ] `youtube_hashtags` are included naturally.
- [ ] `youtube_made_for_kids` is set.
- [ ] `youtube_category` is set.
- [ ] Privacy status is correct for testing or publishing.
- [ ] Comments setting is correct.

## TikTok

- [ ] `tiktok_caption` is present.
- [ ] `tiktok_hashtags` are present and natural.
- [ ] Caption is suitable for public posting.
- [ ] Caption avoids spammy or misleading claims.
- [ ] Check whether any wording may trigger policy or moderation issues.
- [ ] Privacy status is correct for testing or publishing.

## Instagram Reels

- [ ] `instagram_caption` is present.
- [ ] `instagram_hashtags` are present and natural.
- [ ] `instagram_cover_text` is short and suitable for a cover.
- [ ] Video is suitable for Reels.
- [ ] Thumbnail or cover image exists if required by the chosen tool.
- [ ] Posting target is the correct Instagram account connected through OAuth.

## Facebook Reels

- [ ] `facebook_caption` is present.
- [ ] `facebook_hashtags` are present and natural.
- [ ] Video is suitable for Facebook Reels.
- [ ] Confirm whether posting target is a Page or Profile.
- [ ] Posting target is authorized through OAuth or an approved API tool.
- [ ] Privacy status is correct for testing or publishing.
